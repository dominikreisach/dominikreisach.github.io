featured projects

offcut tales

Offcut Tales is a research project on circularity and automation in timber engineering and design. I developed the Rhino3D/Grasshopper plugin Spruce Beetle in C# to streamline the use of waste wood, from computational design to robotic fabrication. The plugin generates offcut positions, joints, and robotic toolpaths. To validate the workflow, I designed and fabricated a proof-of-concept demonstrator structure with a UR10e robot and custom milling end-effector.

compas voxel pack

COMPAS VP is an open-source, object-oriented Python library for packing objects of any shape into containers of any shape. Built on the COMPAS framework, it offers a modular set of packing strategies that can be flexibly adapted to different design and engineering needs. Object orientation and positioning are highly optimized and defined through field-based methods, enabling customizable and efficient workflows for irregular geometries.

robotic mosaics

Robotic Mosaics explores circular design through the reuse of discarded ceramic tiles. Building on COMPAS VP, I developed a workflow where a robot digitizes tiles with an onboard camera, using a computer vision pipeline for shape and color detection. Mosaic designs can be created in a 2D prototype GUI or extended into 3D through Rhino3D/Grasshopper. Fabrication is achieved with a robotic pick-and-place system: tiles are directly placed for 2D mosaics, while 3D mosaics require flipping with an additional pneumatic gripper.
